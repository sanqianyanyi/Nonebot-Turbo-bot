# Nonebot-Turbo-bot

# 简介
Nonebot-Turbo-bot是基于Turbo相关Api文档，提供指令服务的Bot插件。通过该Bot，查询机厅，用户信息等操作。</br>
</br>
# 功能
1.查看当前机厅状态</br>
2.查看Turbo用户信息</br>
懒得列了，看sysallnet_maimai.py有说明</br>
</br>
# 安装
1.NapCatQQ（参考NapCatQQ与Nonebot连接方式文档配置.env即可）</br>
2.Nonebot</br>
3.sqlserver</br>
4.git本仓库代码</br>
5.修改.env中相关管理员QQ号等配置信息</br>
如果不想对接NapCatQQ可自行修改成单Nonebot运行</br>
